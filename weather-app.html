Weather App

See the Weather app deployment...

This application uses Ajax to retrieve the weather forecast for specified city. The app creates a GET request from the user-specified ZIP code to the Open Weather Map REST API, and parses the data on the response to generate and display the current weather conditions and a five-day forecast.

Technologies used: HTML/CSS, JavaScript, AJAX, Heroku.
