Created at the request of (fictional) local business owner, Sarah Dutkiewicz, of (equally fictional) S.D. Industries, this application allows for Ms. Dutkiewicz to create and manage blog posts and other content for her business. By making use of a TinyMCE text editor, she need not use any HTML to create rich text content. All blog posts can be given categories, too. Visitors interested in a certain category can search for it from the home page and see all posts of that category. Anyone visting the blog can see what has been posted, but only users added by an administrator can write new content. And with the help of Spring Security, these new posts will only see the light of day if approved by an administrator.

Demos:

  • View a post from the homepage.
  • Search for posts by category.
  • Log in as a user and write a new blog post.
  • Log in as an Admin and approve the new post.
  • Create a new static page as an Admin.
  • Add a new user as Admin.
  • Additional Admin-level content management.

This application models the interface and processes of a vending machine. Users can "insert" money and select an item for purchase. The system will deplete the purchased item's inventory quantity and calculate and disburse change owed, if any. Users attempting to purchase an out of stock item will be greeted with a message indicating as such, and will be prompted to "insert" a specified amount of money if the item they attempt to purchase is priced at an amount higher than the value of the denomination provided.

Demos:

  • Purchasing an in-stock item with proper funds.
  • Purchasing an item with insufficient funds provided.
  • Purchasing an item that is sold out.

Designed for a fictional flooring wholesaler, this console-based application allows users to manage sales orders for flooring products. On startup, the program will load existing orders from a file. To enter an order, users need only provide some customer information, the purchased product name and quantity, and state location, and the system will automatically calculate the total cost for the product and any applicable installation costs and sales tax. Users can also view, edit, and delete existing orders. Upon exiting the program (or if specifically selected), changes made during that session will be written to the data files. The program also features a training mode, where new users can add and modify existing orders without worrying about altering the live data.

Demos:

  • View all existing orders for a certain date.
  • Add a new order.
  • Edit an existing order.
  • Delete an existing order.

About me

My name is Jeff Wu, and I'm a full stack developer.

I went to THE Ohio State Univerity where I studied business and accounting, and eventually earned my CPA.

My love of programming started when I learned how to use macros to prepare accounting reports in Excel. Seeing what used to take me hours to do be done with the single click of a button made my jaw drop. "Has my computer always been able to do stuff like this?" I asked myself. I began researching coding, eventually enrolling in online classes and building small projects on my own in Java. Craving a little more structure, I enrolled in CSULB's Full Stack Development certificate program. For the better part of a year, it was debits and credits by day, coding by night (later, diaper-changings and bottle feedings worked their way into the rotation), my fascination growing with each carriage return.

Fast-forward to summer 2020, I am now a full stack developer with a variety of completed projects to my name, looking to find a home where I can put my skills to use professionally. It has been an incredible journey so far, and hopefully I am still just beginning.

So please take a resume, send me an email, and take a look at what I have been coding lately, and let me know how I can help you!
